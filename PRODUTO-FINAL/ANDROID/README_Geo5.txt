Notas sobre apk Geo5:

-A aplicação utiliza uma funcionalidade que guarda uma sessão do utilizador para evitar estar semnpre a fazer login. No entanto, se a sessão de login expirar sem o utilizador fazer logout, o utilizador não vai conseguir fazer nehuma operação. Para corrigir basta fazer logout e login outra vez.
- A remoção de caminhos é apenas local.
- O upload de fotos no caminho pela aplicação só funciona localmente, no momento.
